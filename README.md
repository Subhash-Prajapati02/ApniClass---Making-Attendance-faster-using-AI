# 🚀 ApniClass – AI Attendance System

ApniClass is a full-stack AI-powered attendance system that automates classroom attendance using Face Recognition and Voice Verification.

## 🔥 Features
- 👨‍🏫 Teacher Dashboard (Manage subjects, attendance)
- 👨‍🎓 Student Portal (Join via link/code)
- 🤖 Face Recognition Attendance
- 🎤 Voice-based Attendance
- 🔗 QR / Link-based Enrollment
- 📊 Attendance Tracking Dashboard

## 🛠 Tech Stack
- Frontend: Streamlit
- Backend: Supabase (PostgreSQL)
- AI: dlib, scikit-learn
- Language: Python

## 🌐 Live Demo
👉 https://apniclass-subhash.streamlit.app

## 🧠 How it Works
1. Teacher creates subject
2. Shares join link
3. Students enroll
4. Face/Voice registered
5. AI marks attendance automatically
